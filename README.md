# student-performance-analysis

## Objective
To analyze student performance using Excel, SQL, and Power BI.

## Tools Used
- Excel
- SQL (MySQL)
- Power BI
- Git & GitHub

## Project Structure
- excel/ → Data cleaning and analysis
- sql/ → SQL queries and database work
- powerbi/ → Dashboard files

## Dataset Details
The dataset contains:
- Student names
- Subject marks
- Attendance percentage
- Gender information

## Excel Analysis
- Created average marks column
- Built performance comparison chart
- Analyzed attendance vs performance
- Identified top-performing students

## SQL Analysis
- Created database and student table
- Inserted student records using SQL
- Calculated average marks using queries
- Performed gender-wise performance analysis
- Filtered students based on attendance

## Key Insights
- Students with higher attendance generally scored better
- Female students showed higher average performance
- Priya was the top-performing student in the dataset
- Low attendance students had weaker overall scores
