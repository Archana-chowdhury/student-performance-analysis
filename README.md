# Student Performance Analysis

## Objective
To analyze student performance using Excel, SQL, and Power BI and identify key patterns in academic performance and attendance.

---

## Tools Used
- Excel
- SQL (MySQL)
- Power BI Desktop
- Git & GitHub

---

## Project Structure
- excel/ → Data cleaning, preprocessing, and initial analysis
- sql/ → Database creation and SQL queries for analysis
- powerbi/ → Interactive dashboard and visualizations

---

## Dataset Details
The dataset contains student academic information including:
- Student ID and Name
- Gender
- Subject marks (Math, Science, English)
- Attendance percentage

---

## Excel Analysis
- Created average marks column for each student
- Built performance comparison charts
- Analyzed attendance vs performance relationship
- Identified top-performing students
- Performed basic data cleaning and formatting

---

## SQL Analysis
- Created database and student table
- Inserted student records into MySQL
- Calculated average marks using SQL queries
- Performed gender-wise performance analysis
- Filtered students based on attendance criteria
- Used ORDER BY and GROUP BY for insights

---

## Power BI Dashboard
- Built interactive dashboard using Power BI Desktop
- Visualized student performance across subjects
- Analyzed relationship between attendance and marks
- Created KPI card to show top performer (max average score)
- Compared gender-wise performance using bar charts
- Used scatter plot for correlation between attendance and performance

---

## Key Insights
- Students with higher attendance generally performed better
- Female students showed slightly higher average performance
- Priya was the top-performing student in the dataset
- Low attendance students tended to have lower academic scores
- Attendance shows a positive correlation with performance

---

## Future Improvements
- Add larger real-world datasets for deeper analysis
- Build advanced SQL queries for complex insights
- Improve Power BI dashboard with filters and slicers
- Add predictive analytics using Python or machine learning

---

## Conclusion
This project demonstrates an end-to-end data analysis workflow using Excel, SQL, and Power BI. It highlights how raw data can be transformed into meaningful insights for academic performance evaluation.
